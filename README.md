# Movie Prediction System

## Overview
The **Movie Prediction System** is a machine learning project that predicts movie performance using historical movie data. The project uses data analysis and predictive modeling techniques to identify patterns that influence movie success.

---

## Objectives
1. Analyze historical movie dataset  
2. Identify key factors affecting movie performance  
3. Build machine learning models for prediction  
4. Evaluate model accuracy and performance  

---

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Jupyter Notebook  

---

## Dataset
The dataset contains information related to movies such as:

- Movie Title  
- Genre  
- Budget  
- Revenue  
- Ratings  
- Other relevant attributes used for prediction  

---

## Process Flow
1. Data Collection  
2. Data Cleaning and Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Selection  
5. Model Building  
6. Model Evaluation  
7. Prediction Results  

---

## Machine Learning Models Used
- Linear Regression  
- Random Forest Regressor  
- Support Vector Regression (SVR)  

---

## Results
The models are trained on historical movie data to predict movie performance. Evaluation metrics are used to compare the models and determine the best-performing algorithm.
